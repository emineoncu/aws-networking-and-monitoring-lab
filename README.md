# AWS Networking and Monitoring Lab

Hands-on AWS project simulating real-world Cloud Support tasks involving networking, infrastructure setup, and monitoring.

VPC configuration, traffic control, and system monitoring using AWS tools.

---

## Overview
This project demonstrates key AWS cloud support skills related to networking and monitoring.

I built a Virtual Private Cloud (VPC), configured subnets and routing, and implemented monitoring using CloudWatch and CloudTrail.

---

## Architecture

- VPC → isolated cloud network  
- Subnets → public and private segmentation  
- Internet Gateway → enables public internet access  
- NAT Gateway → allows private resources to access the internet securely  
- CloudWatch → monitoring and alerting  
- CloudTrail → logging and auditing AWS activity  

---

## What I Built
- Created a VPC with public and private subnets
- Configured route tables and internet gateway
- Set up NAT Gateway for outbound internet access
- Implemented CloudWatch metrics, logs, and alarms
- Enabled CloudTrail for tracking user activity
- Tested connectivity between network components
